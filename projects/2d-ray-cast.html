2D Ray Casting: Light

This is a 2D ray-casting demo using p5.js. I undertook this project as a method to learn more about the graphics technique as well as teach myself some javascript.

Controls

  • Mouse Drag - Adds a wall
  • Mouse Wheel - Toggles Line/Draw mode
  • Key 'R' - Adds a random wall
  • Key 'C' - Removes all the walls (Except outer boundaries)
  • Key 'D' - Removes most recently added wall
  • Key 'Q' - Toggles Line/Draw mode

My Reponsibilites

I was responsible for this entire project from planning to full implementation.

Try Me!
