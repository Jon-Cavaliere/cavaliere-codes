CG2 Tool

    This software is for creating and editing crosswords.

It was developed as a utility for a game I am creating. The current iteration is the 3rd full update. It has an option for loading a list of words from disk, and allows the user to place them manually, or have an algorithm automatically place all of them. The libraries it utilizes are pygame and tkinter.

Features

  • Visual Representation of puzzle
  • File Saving / Loading
  • Dataset Loading (Lists of words)
  • Auto-Add - Algorithm automatically add a word where it fits
  • Manual Adding (Horizontal or Vertical)
  • And much more!
