Mandelbrot Set

This was a neat project that I wrote in C++ using my own game engine. It uses OpenGL for rendering and GLFW for event handling and windowing. I wanted to see what my engine was capable of, so I wrote this. It was basically a stress test. The actual code for calculating the mandelbrot set shapes was done in a vertex and fragment shader.
